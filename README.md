# Tg Bot CGI-like Server

## Предварительные требования
1. Зарегистрируйте бота в телеграм с помощью [@BotFather](https://t.me/BotFather)
2. Убедитесь, что на ПК установлен Python. Сделать это можно выполнив команду в терминале
    ```shell
    python --version
    ```
    Если видите ошибку вместо номера версии, попробуйте команду     
    ```shell
    python3 --version
    ```
    Если сработала вторая команда -- используйте её запуска кода в дальнейшем 
3. Если вы используете Windows и Python не установлен -- скачайте установочный файл 
[по ссылке](https://www.python.org/downloads/) и установите Python

## Инструкция по установке и запуску

### 1 Настройка токена
Замените текст `1111111111:YOUR-TgBotToken` в файле `config.ini` на ваш токен, полученный в [@BotFather](https://t.me/BotFather)

### 2. Запуск бота (в отдельном терминале)
Находясь в папке проекта выполните команду
```bash
python server.py
```
Замените `python` на `python3`, если Python уже установлен, но не запускается из командной строки. В первую очередь эта
особенность применима для Linux и MacOS

## Как изменить поведение бота
- Весь алгоритм обработки и отправки сообщений Телеграм необходимо описать в модуле `bot.py`
- Бот работает в формате CGI-like сервера, т.е. код модуля `bot.py` запускается в ответ на любое сообщение пользователя
- Сообщение пользователя, инициирующее запуск бота хранится в `sys.argv[1]`
- Чтобы отправить ответ пользователю, необходимо вывести сообщение с помощью функции `print("Ваше сообщение")`
